Function parse_entries 

pub fn parse_entries(args: &Value) -> Result<Vec<PlanEntry>, Error>
Expand description

Parse and normalize the todos argument array into PlanEntry values. Shared by the tool’s execute and by the tool-execution layer’s TurnEvent::Plan emission so the shape is validated once.

Rules: content required; status required and must be one of pending/in_progress/completed; unknown priority coerces to medium; activeForm optional. An empty todos array is a valid clear.
